## 各ファイルのコンパイル方法
- C言語
  ```
  cl filename.c
  ```
- C言語＋OpenMP
  ```
  cl filename.c -openmp
  ```
- CUDA
  ```
  nvcc filename.cu -o filename.exe
  ```
